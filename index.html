Lectionary Readings Parser

About:


When Sundays and Seasons displays lectionary readings, it makes the verse numbers tiny superscript in a light-colored font. But when you copy the text and paste it into a bulletin, PowerPoint slide, or music leader's folio, the verse numbers become full font, rendering4 text that5 looks6 like this7a.


This application will parse text copied from the Sundays and Seasons "Readings in Full" page for a single Sunday. It will output a formatted group including the Prayer of the Day, the First Reading, the Psalm (either chanted or spoken, depending on your choice), the Second Reading, and the Gospel. It removes verse numbers and checks, making it more easily readable by liturgists and better for powerpoint presentations.


Instructions:


Go to sundaysandseasons.com and sign up or sign in to your account. Use the calendar to navigate to the Sunday you wish to extract texts from, and click "Day Texts," and "Read in Full." Then select all the windows content using Ctrl-A, copy it using Ctrl-C, and then navigate to this page and use Ctrl-V to paste it into the input box. "Psalm is chanted" is selected by default; deselect it if you want the parser to remove the flexas. Select "Sunday in Pentecost" if this Sunday is between Trinity Sunday and the first Sunday of Advent (S&S changes the way they display readings during that time and it causes the parser to add extra text if you don't check it then). Finally, make sure you put the required Sundays and Seasons license number and copyright notice on any documents you prepare using this output.
